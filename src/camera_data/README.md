# 镜头参数数据目录

此目录用于存储各类镜头的参数数据：

- 镜头标定数据
- 畸变校正参数
- 镜头规格参数
- 相机配置文件

## 数据格式
- 标定数据：.json, .yaml
- 参数文件：.xml, .ini
- 配置文件：.config

## 目录结构
```
camera_data/
├── calibration/    # 标定数据
├── parameters/     # 镜头参数
└── configs/        # 配置文件
```

## 使用说明
1. 标定数据需要包含完整的矩阵信息
2. 参数文件需要注明适用的镜头型号
3. 建议添加参数说明文档
4. 重要参数建议做好备份
